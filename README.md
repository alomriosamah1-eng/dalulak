# دليلك التعليمي والعملي

موقع ويب عربي تفاعلي (SPA) يجمع ويُنظّم الموارد التعليمية المجانية، الأدوات الإنتاجية، منصات العمل الحر، قنوات اليوتيوب التعليمية، واختبارات التقييم في مكان واحد.

**هدية من المهندس أسامة محمد علي سعيد العمري**  
هاتف: 711666863

---

## 🚀 الميزات

- **نظام الصفحة الواحدة (SPA)** — التبديل بين الأقسام بدون إعادة تحميل الصفحة
- **تصميم زجاجي (Glassmorphism)** — واجهة عصرية مع تأثيرات blur
- **الوضع الداكن/الفاتح** — حفظ التفضيل في LocalStorage
- **بحث فوري** — في القسم الحالي مع debounce
- **المفضلة** — حفظ وإدارة محلية مع عداد
- **شاشة ترحيب** — رسوم متحركة تفاعلية
- **نوافذ منبثقة (Modals)** — عرض تفاصيل كل مورد
- **الإشعارات (Toasts)** — Bootstrap Toasts
- **تصدير/استيراد بيانات** — وحدة Dashboard تدعم JSON
- **شريط إجراءات سفلي** — للأجهزة المحمولة
- **زر رجوع للأعلى** — Smooth scroll
- **تجاوب كامل** — 5 نقاط قطع للشاشات المختلفة

## 🔧 آلية العمل

1. **index.html** يحمل جميع الـ Scripts بترتيب: `main.js` ← `dashboard.js` ← ملفات البيانات
2. **main.js** يهيئ التطبيق: ضبط الوضع اللوني → تحميل البيانات → إعداد الأحداث → عرض شاشة الترحيب
3. ملفات البيانات (`*-data.js`) تستدعي `window.updatePlatformData()` لتغذية `platformData`
4. `renderCards(section)` ينشئ بطاقات Bootstrap لكل عنصر
5. `switchSection(section)` يتحكم في إظهار/إخفاء الأقسام
6. البحث يتم في القسم الحالي عبر تصفية `platformData[currentSection]`
