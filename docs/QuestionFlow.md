# تدفّق الأسئلة (Question Flow)

> إصدار: 1.0  
> آخر تحديث: يوليو 2026  
> الحالة: **تصميم أولي**

---

## 1. مبادئ التصميم

1. **الودّ أولاً** — كل سؤال يبدأ بجملة حوارية ودودة، لا جافّة.
2. **البطاقة الواحدة** — سؤال واحد فقط لكل شاشة (Focus).
3. **التقدّم المرئي** — شريط تقدم يظهر في الأعلى.
4. **التحكّم** — المستخدم يمكنه العودة للسؤال السابق أو التخطي (إن مسموح).
5. **الحفظ التلقائي** — كل إجابة تُحفظ فوراً.

## 2. القسم الأول: المعلومات الأساسية

### 2.1 السؤال 1 — العمر

```
النص: "كم عمرك؟ 🎂"
النوع: اختيار من متعدد (قائمة منسدلة أو أزرار)
الخيارات: أقل من 15 | 15–18 | 19–24 | 25–30 | 31–40 | 41–50 | أكثر من 50
إلزامي: ✅
المعرف: age
```

### 2.2 السؤال 2 — المرحلة الدراسية

```
النص: "ما هي مرحلتك الدراسية الحالية؟ 📚"
النوع: اختيار من متعدد (أزرار)
الخيارات: ثانوي | جامعي (بكالوريوس) | دراسات عليا (ماجستير/دكتوراه) | خريج | موظف | أخرى
إلزامي: ❌ (اختياري)
المعرف: educationLevel
```

### 2.3 السؤال 3 — الدولة

```
النص: "من أي بلد أنت؟ 🌍"
النوع: اختيار من متعدد (قائمة منسدلة) + بحث
الخيارات: قائمة بجميع الدول العربية + دول عالمية رئيسية
إلزامي: ❌
المعرف: country
```

### 2.4 السؤال 4 — اللغة المفضلة

```
النص: "بأي لغة تفضل التعلم؟ 🌐"
النوع: اختيار من متعدد
الخيارات: العربية | الإنجليزية | كلتاهما
إلزامي: ✅
المعرف: preferredLanguage
```

### 2.5 السؤال 5 — مستوى اللغة الإنجليزية

```
النص: "ما هو مستواك في اللغة الإنجليزية؟"
النوع: اختيار من متعدد (بطاقات مع وصف)
الخيارات:
  - مبتدئ: "أفرد جمل بسيطة"
  - متوسط: "أفهم المحادثات والمقالات"
  - متقدم: "أتحدث بطلاقة وأقرأ التقارير"
  - محترف: "مستوى اللغة الأم"
إلزامي: ❌ (اختياري — لكن تظهر في النتائج)
المعرف: englishLevel
ملاحظة: إذا اختار "العربية" فقط في السؤال السابق، يُتخطى هذا السؤال.
```

### 2.6 السؤال 6 — الخبرة التقنية

```
النص: "ما هي خبرتك في المجال التقني بشكل عام؟ 💻"
النوع: اختيار من متعدد (بطاقات)
الخيارات:
  - مبتدئ تماماً: "لم أبرمج من قبل"
  - معرفة بسيطة: "جربت بعض المفاهيم"
  - متوسط: "أستطيع بناء مشاريع بسيطة"
  - متقدم: "أعمل في المجال التقني"
  - محترف: "خبير في تخصصي"
إلزامي: ✅
المعرف: techExperience
```

## 3. القسم الثاني: الهدف الرئيسي

### السؤال 7 — أهدافك

```
النص: "ما الذي تريد تحقيقه؟ حدد أهدافك 🎯"
النوع: اختيار متعدد (مجموعة بطاقات مع أيقونات)
إلزامي: ✅ (حدد هدفاً واحداً على الأقل)
المعرف: goals

الخيارات:
  🖥️  تعلم البرمجة
  🤖  تعلم الذكاء الاصطناعي
  💼  الحصول على وظيفة
  🏠  العمل الحر (Freelancing)
  🚀  إنشاء شركة
  🌱  تطوير الذات
  🎨  تعلم التصميم
  📈  التسويق الرقمي
  📊  تحليل البيانات
  🔒  الأمن السيبراني
  🌐  تعلم اللغات
  📋  إدارة الأعمال
  🎬  صناعة المحتوى
  🛒  التجارة الإلكترونية
```

المستخدم يستطيع اختيار أكثر من هدف.  
النظام يأخذ الأهداف بوزن متساوٍ ثم يعطي أولوية حسب باقي الأسئلة.

## 5. القسم الرابع: الخبرة الحالية

### السؤال 9 — مستواك في كل مجال

```
النص: "في كل مجال من المجالات التي اخترتها، ما مستوى خبرتك الحالي؟ 📊"
النوع: بطاقة لكل مجال يختاره المستخدم مع شريط تصنيف
إلزامي: ✅
المعرف: skillLevels

لكل مجال، يعرض:
  🔵 مبتدئ تماماً
  🟢 لدي معرفة بسيطة
  🟡 متوسط
  🟠 متقدم
  🔴 محترف
```

الأسئلة تُنشأ ديناميكياً بناءً على إجابات السؤال السابق.  
يستخدم آلية **Dynamic Questions** — لو اختار 3 مجالات فقط، يظهر له 3 أسئلة فقط.

## 10. شاشة النهاية

بعد السؤال الأخير، لا تظهر النتائج مباشرة.

تظهر شاشة "جاري بناء رحلتك التعليمية..." مع رسوم متحركة، ثم تنتقل إلى الصفحة الرئيسية المخصصة.

انظر [SmartOnboarding.md](./SmartOnboarding.md) ← القسم 4 لمزيد من التفاصيل.

## 11. إدارة الأسئلة من لوحة الإدارة

| الخاصية | الطريقة |
|---------|---------|
| إضافة سؤال | POST /api/admin/questions |
| تعديل سؤال | PUT /api/admin/questions/:id |
| ترتيب الأسئلة | PATCH /api/admin/questions/reorder |
| تحديد الإلزامي | الحقل `required: boolean` |
| تحديد الوزن | الحقل `weight: number` |
| تعطيل سؤال | الحقل `active: boolean` |

كل سؤال يُخزّن بهيكل:

```json
{
  "id": "q_age",
  "section": 1,
  "order": 1,
  "type": "single_choice",
  "required": true,
  "question": {"ar": "كم عمرك؟", "en": "How old are you?"},
  "options": [
    {"value": "under_15", "label": {"ar": "أقل من 15", "en": "Under 15"}, "weight": 0},
    ...
  ],
  "skipLogic": { "dependsOn": "preferredLanguage", "skipIf": "arabic_only" },
  "active": true,
  "createdAt": "",
  "updatedAt": ""
}
```
